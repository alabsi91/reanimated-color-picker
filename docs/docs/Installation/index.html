Installation

Prerequisites​

  • react-native-reanimated version 2.0.0 or higher.​

  • react-native-gesture-handler version 2.0.0 or higher.​

For Expo users

Expo managed workflow version 44 or higher is required.​

Installation​

  • Open a Terminal in the project root and run:
  • npm
  • yarn
npm i reanimated-color-picker
yarn add reanimated-color-picker
