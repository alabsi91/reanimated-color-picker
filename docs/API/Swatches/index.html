<Swatches />

  • A list of color swatches is utilized for efficient color selection.

Props​

colors​

  • Custom swatches colors.
  • type: string[]
  • default: material colors

style​

  • Swatches container style.
  • type: ViewStyle

swatchStyle​

  • Swatch style.
  • type: ViewStyle
note
  • Certain style properties will be overridden.
