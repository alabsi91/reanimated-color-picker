# `<BrightnessSlider />`

![brightness](@assets/images/brightness.png)

A slider for adjusting the color's brightness channel in the HSV color space.

## Props

### `adaptSpectrum`

The slider background color spectrum adapts to changes in saturation and brightness.

`type: boolean` · `default: false`
