# `<Swatches />`

![swatches](@assets/images/swatches.png)

A list of color swatches for quick color selection.

## Props

### `colors`

The swatch colors. Defaults to Material Design colors.

`type: string[]` · `default: material colors`

---

### `style`

The swatches container style.

`type: ViewStyle`[^style-override]

---

### `swatchStyle`

The individual swatch style.

`type: ViewStyle`[^style-override]

---

[^style-override]: Certain style properties will be overridden.
