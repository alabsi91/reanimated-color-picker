# Installation

## Prerequisites

Before installing, make sure you have the following packages set up:

- [react-native-reanimated](https://docs.swmansion.com/react-native-reanimated/docs/fundamentals/getting-started/#installation) `>= 2.0.0`.
- [react-native-gesture-handler](https://docs.swmansion.com/react-native-gesture-handler/docs/fundamentals/installation) `>= 2.0.0`.
- For Expo users: [Expo managed workflow](https://docs.expo.dev/get-started/installation/) `>= 44`.

## Install

Run the following command in your project root:

```bash
npm install reanimated-color-picker
```
